JSON
JSON(Javascript Object Notation) is exchange format readable by humans and alothough it is 
based on Javascript, it  islanguage-indepenedent ((1)https://www.mongodb.com), which 
means that JSON can be used with other programming languages ((2)https://learn.microsoft.com).

BSON
BSON stands for Binary JSON((1)https://www.mongodb.com), which is a serialized version of a JSON document,
which menas it not human readable like JSON and BSON is used within MongoDB ((3)https://www.geeksforgeeks.org)
